¿Por qué usar Linux?

Fecha de Publicación: 13 de diciembre de 2025

Linux per se es un kernel creado por Linus Torvalds, es el actual núcleo de la mayoría de servidores a nivel mundial. Pero, Linux no fue diseño exclusivamente para servidores, si no que, también puede ser usado como Sistema Operativo de Escritorio, es decir, ser usado por las personas comunes para tú o como yo.

"Una computadora es como el aire acondicionado. Se vuelve inútil una vez que abres Windows. - Linus Torvalds"

¿Qué distribución usar?

Es muy común y fácil iniciar en Linux mediante una buena distribución que sea fácil y amigable con el usuario, ya que, vas a aprender algo nuevo y es necesario que la capa de introducción sea lo menos espesa posible, por eso creo fielmente que la mejor opción para iniciar es con Linux Mint.

Linux Mint está basado en Ubuntu, que a su vez está basado en Debian. Todas estas distribuciones usan un gestor de paquetes para administrar toda la distribución y su software.
Mira un ejemplo de código para su gestor de paquetes APT:
sudo apt update && sudo apt upgrade

Al adentrarte al mundo de Linux, te encontrarás con la terminal, aquella cosa que parece solo para técnicos y programadores especializados, ya que, al venir de Windows, jamás la había usado a excepción de comandos que veías por Internet para mejorar el rendimiento o limpiar la caché. Si te vas a adentrar al mundo de la terminal, conocerás Bash, déjame darte un pequeño código shell en Bash (su configuración): 

            
              #
              # ~/.bashrc
              #

              # If not running interactively, don't do anything
              [[ $- != *i* ]] && return

              alias ls='ls --color=auto'
              alias grep='grep --color=auto'
              PS1='[\u@\h \W]\$ '
              . "$HOME/.cargo/env"
