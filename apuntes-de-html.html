¿Qué es HTML?

Sus siglas signfican Hyper Text Markup Language

HTML es el lenguaje de marcado estándar para crear páginas web.

En la Web, CSS se usa para la presentación

En la Web, JavaScript se usa para la interactividad

HTML se encargade darle formato y describir el contenido de la Web

Dentro de HTML, el < h1 > es la etiqueta de apertura mientras que el < /h1 > es la etiqueta de cierre. Además de que, cuando escribimos HTML, no se muestran las etiquetas, ya que, el navegador las interpreta.

Una etiqueta es < h1 > < / h1 > mientras que un elemento es todo < h1 >contenido< / h1 >

Nota Importante: Las etiquetas deben cerrarse en el orden inversor en el que se abrieron.
Por ejemplo: Texto importante


Atributos HTML

Son valores que se le pueden dar a las etiquetas para que tengan ciertas características. Por ejemplo en un img con un src="lugar_de_la_imagen"

Nota Curiosa: En HTML, no se necesitan las comillas si el valor no tiene espacio, por ejemplo:

  • img title=gato src="imagen.jpg"
  • input type=text value=Hola

Aunque se puedan omitir las comillas, es recomendable usarlas para mayor claridad y legibilidad.

También podemos encontrar a ID y Class que son atributos que se usan para identificar elementos.

  • ID: Es un identificador único, es decir, no puede haber dos elementos con el mismo ID.
    Por ejemplo: Si asignamos a un img id="foto-yo", al ya existir un ID llamado foto-yo, ningún otro elementos puede hacer uso de ese ID.
  • Class: Permite clasificar varios elementos bajo un mismo nombre. Por ejemplo: Podemos asignar un class="list-items" a varios elementos y usar una misma clase para referirnos a ellos.
  • Role: Permite especificar el propósito de un elemento. Muchos elementos como header y article ya tienen un role definido por defecto. Por ejemplo: Podemos asignar un role="button" a un elemento para indicar que es un botón. Algunos de los roles que podemos implementar son:
    • alert: Para mostrar mensajes urgentes
    • dialog: Para mostrar un diálogo modal
    • navigation: Para definir una barra de navegación
  • Como nota, lo ideal es usar un elemento HTML ideal, en vez de usar un div con role="button" mejor usar directamente el elemento button
Nota Importante: Los atributos como class, id, style funcionan con cualquier elementos. Mientras que src, alt, title, hidden, checked, etc. funcionan con ciertos elementos.

Atributos Booleanos

Son atributos que no necesitan valor, por ejemplo: hidden o checked Ejemplo de uso:

  • img hidden src="imagen.jpg"
    • hidden hace que la imagen no se muestre pero sigue existiendo, además de que con estos atributos, el mero hecho de aparecer ya aplican su propósito, es decir, no es necesario poner hidden="true" ni nada de eso.
  • input type=checkbox checked

Case Sensitive en HTML

HTML no es case sensitive, es decir, no pasa nada al poner un IMG en vez de un img.
Aunque a día de hoy, se recomienda altamente usar minúsculas por los siguientes motivos:

  • Consistencia: Se mantiene un código consistente y legible, fácil de leer y analizar.
  • Estándar Actual: Las normas actuales de la web recomiendan altamente usar minúsculas, ya que es una muy buena práctica el hecho de no combinar más de 2 tipos de letras; solo minúsculas.

Etiquetas

  • Strong: Se usa para marcar y resaltar texto importante en negritas
  • Em: Se usa para marcar un énfasis ligero y menos pesado que con "strong" dándole un toque cursivo.
  • Article: Es un bloque de contenidos para marcar artículos dentro de la página web, puede existir de forma independiente
  • Div y Span: Son etiquetas muy útiles pero no son semántica y por lo tanto habría que regular su uso para implementarlas correctamente. Div: Sirve para agrupar bloques de contenidos y poder aplicarle estilos y scripts de manera más rápida y sencilla Span: Sirve para agrupar bloques de contenidos en línea que necesitan estilos específicos

Enlaces Especiales

En HTML, hay pequeños elementos que podemos usar que van más allá del protocolo HTTP mediante un a href, como los que veremos a continuación

  • mailto: Sirve para crear enlaces que automáticamente abran el cliente de correo del usuario Por ejemplo: "< a href="mailto:emailgenerico@gmail.com>Envíame un correo< /a>"
  • tel: Redirige a una app para hacer llamadas telefónicas con el número proporcionado Por ejemplo: "< a href="tel:+521234567899">LLama ahora< /a>"
